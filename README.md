## Installation

This repo uses a lot of npm packages so im not going to bother listing them all, however you can install them all with the command below as with all node projects!

```npm i``` or ```npm install```

## Usage

To run this project locally you can use the command listed below or you can easily deploy it to vercel for free.

```npm run dev``
